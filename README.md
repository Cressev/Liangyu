# Detection Project Memory

This directory is the working project root for the detection setup on the training machine.

Key files:
- `AGENTS.md`: durable rules and path conventions.
- `CURRENT.md`: active state and next steps.
- `PROGRESS.md`: project storyline and completed milestones.
- `log.md`: chronological work log.
- `findings/`: durable environment/workflow findings.
- `requirements_liangyu.txt`: Python package requirements from the local configuration bundle.
