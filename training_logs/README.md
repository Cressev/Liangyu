## Notes

- Smoke run directories are under /dpc/yuanxiangqing/projects/detection/MutilModel_423/run_smoke and smoke logs are under /dpc/yuanxiangqing/projects/detection/training_logs/smoke.
- Formal run directories are kept outside run_smoke: YOLO under run_yolo_bs64, RT-DETR under run_non_yolo_bs64, and SSD/Faster R-CNN under run_non_yolo_legacy.
- Legacy SSD/Faster R-CNN eval callbacks were patched on 2026-04-26 to put temporary map files under each run's log directory instead of shared `.temp_map_out`, preventing concurrent RGB/Sonar eval cleanup collisions.
- The replacement ultralytics.zip does not write mapS/mapM/mapL to results.csv, but it includes COCOevalBBoxMM. AP and size metrics were recomputed from each best.pt via /dpc/yuanxiangqing/projects/detection/eval_size_maps.py and saved under /dpc/yuanxiangqing/projects/detection/reports/size_maps/.
- Launch state: /dpc/yuanxiangqing/projects/detection/training_logs/yolo_bs64_launch_20260426_085016.tsv
